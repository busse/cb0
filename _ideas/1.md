## Overview

This idea represents a typical project or concept that you might want to track. Ideas are the top-level organizational unit in the system.

## Stories

Each idea contains multiple stories (specific tasks or features). These are organized as:
- i1.s0 - The intent behind this idea
- i1.s1 - A sample implementation story

Stories can be assigned to sprints and tracked through updates.

## Usage

To create your own ideas:

1. Add a new file in `_ideas/` with the idea number as the filename
2. Create a corresponding directory in `_stories/` for that idea's stories
3. Add stories as needed
4. Assign stories to sprints to plan your work
